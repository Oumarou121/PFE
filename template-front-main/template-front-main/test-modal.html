📋 Instructions de Test

  1. Assurez-vous que l'application Angular est démarrée sur http://localhost:4200
  2. Connectez-vous avec les identifiants : admin@test.com / admin123
  3. Naviguez vers la page Processus dans le menu
  4. Testez les fonctionnalités suivantes :

🔧 Fonctionnalités à Tester

✅ Ajout de Processus

  • Cliquer sur "Nouveau processus"
  • Remplir le formulaire
  • Sélectionner un pilote
  • Ajuster le score de conformité
  • Sauvegarder
✓ Modal doit s'ouvrir et permettre la création

✏️ Modification de Processus

  • Cliquer sur une carte de processus
  • Ou utiliser le bouton "Modifier" (icône crayon)
  • Modifier les informations
  • Sauvegarder les changements
✓ Modal doit s'ouvrir avec les données pré-remplies

🗑️ Suppression de Processus

  • Cliquer sur le bouton "Supprimer" (icône poubelle)
  • Confirmer la suppression
✓ Modal de confirmation doit apparaître

🔔 Messages Toast

  • Après chaque action (création, modification, suppression)
  • Messages de succès ou d'erreur
✓ Toast doit apparaître en bas à droite

🐛 Problèmes Courants et Solutions

⚠️ Si les modales ne s'ouvrent pas :

  • Vérifiez la console du navigateur (F12) pour les erreurs
  • Assurez-vous que Angular Material est correctement importé
  • Redémarrez le serveur de développement

❌ Si les données ne se sauvegardent pas :

  • Vérifiez que le serveur JSON (port 3001) fonctionne
  • Contrôlez les appels API dans l'onglet Network
  • Vérifiez les logs de la console
